# app.py — Correr el servidor

## Requisito previo
Instalar Flask una sola vez antes de correrlo por primera vez:
```
pip install flask
```

## Comando para correr
Entrar a la carpeta Proyecto y ejecutar:
```
cd Proyecto
python app.py
```

## Cómo usarlo
1. Correr el comando de arriba
2. Abrir el navegador en: http://127.0.0.1:5001
3. Para detener el servidor: `Ctrl + C` en la terminal
